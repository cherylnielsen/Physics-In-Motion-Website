Physics in Motion

Graduate Project for MS in Computer Science, San Francisco State University

by Cheryl Nielsen

Introduction

Many students drop out of classes in Physics and Engineering due to the high level of difficulty in understanding these topics. The online tutorials available are frequently static and unsatisfactory. The typical tutorial usually consists of reading material (like a textbook with hyperlinks) or videos (classroom lectures or simple animations). While this can be helpful, it still relies on the traditional rote memorization of a problem type and solution formula. I believe that a different approach is needed to make the material easier to understand.

I propose a project that uses today’s powerful interactive online 3D gaming technologies to create a new type of online science tutorial. Not for a game, but for the creation of a truly visual, immersive, and hands on learning experience. This will take the form of tutorial labs that let the student explore the topics being learned as a fully interactive experience.

Project Concept

This project consists of two parts: a website for supporting infrastructure, and the tutorial labs in Unity. The website has five main parts: a home page, tutorial lab information pages, student pages, Professor pages, and wiki pages. The home page has a brief summary of the project and its goals, project information links, a list of tutorial labs currently available, and login/logout. The tutorial lab information pages will have a description of the lab activity, a list of topics to be covered, prior knowledge required for the lab, and lab ratings. The wiki pages will contain project reports and documentation of the project code.

The student and Professor web pages will provide multiple services. The student pages include lists of assigned and completed tutorial labs, links to send a notice of lab completion to a Professor, and links to download lab summaries, data, graphs, etc. The Professor pages include lists of assigned tutorial labs, which labs each student has completed or incomplete, and a form to assign labs to students. The Professor will also have links to download lab summaries, and links to download a list of frequent problems encountered by students for a particular tutorial lab.

Three or four tutorial labs will be developed. They are Linear Motion and Collisions, Beginning RC Circuits, Vector Calculus of 3D Volumes and Surfaces, and the Photoelectric Effect from Quantum Mechanics. These tutorial labs will take full advantage of the power of 3D gaming technology available today. Each tutorial lab will be animated and fully interactive to make the topics more approachable, interesting, and informative, instead of intimidating and dry. In the tutorial labs the student will perform hands-on interactive experiments and activities directly related to the concepts they are trying to learn. Since the project is in 3D, the student will be able to move around to view lab events and graphs from multiple angles. Measurements in the tutorial labs will be collected electronically, as if in a standard industrial lab. The student

will be able to vary many parameters in each lab so they can observe the causes and effects of theirinteractions. Additional animations and closeups of what’s happening will be provided to enhance the student’s understanding. The student will then analyze the collected data with interactive graphs and math calculations done inside the tutorial. At the end of a tutorial lab, a summary will be provided including data, graphs, equations, and problems encountered.

Along the way, the automated mentor will explain topics, give hints, and provide corrections as needed. If the student asks for help during the tutorial lab, the automated mentor will highlight problem areas and offer hints. For math calculations, the automated mentor looks for errors as the student solves the equations. These errors, problems, and hints are later documented as reminders in the lab summary. If enough students encounter the same problem, error, or hint in a tutorial lab, it will be reported to the Professor as a topic that may need more help in the classroom.

Technologies

This project will involve the full stack development of a website, game server, interactive 3D graphics, and a secure database. All parts will be developed in a modular plug and play format. This will enable the code to have a life after the project is over, by enabling changes in implementation and the addition of more tutorial labs.

Client Side

  • The web pages will be made with HTML 5, CSS 3, and JavaScript/jQuery.
  • The tutorial labs will be made with Unity 2018 and Visual Studio C# 2017.
  • The graphics will be made with Unity 2018, Adobe Creative Suite 2018, and Autodesk Maya 2018.

Server Side

  • Server code will be PHP 7 for the web pages and Java 10 for the tutorial labs.
  • MS SQL Server 2017 will be the database for the tutorial labs.
  • MySQL 8 will be the database for the web site.
  • AWS will be the cloud service host for both the web pages and the tutorial labs.
